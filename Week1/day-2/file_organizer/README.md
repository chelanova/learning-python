# File Organizer

Program sederhana untuk organize files berdasarkan tipe.

## Cara Pakai

1. Copy `organizer.py` ke folder yang mau diorganize
2. Jalankan: `python organizer.py`
3. Ketik `y` untuk mulai

## Kategori

- Images: jpg, jpeg, png, gif, bmp
- Documents: pdf, doc, docx, txt
- Videos: mp4, avi, mkv
- Music: mp3, wav
- Archives: zip, rar, 7z
- Others: semua file lainnya

## Yang Dipelajari

- File operations dengan os module
- Loop dan conditions
- Dictionary untuk mapping
- Move files dengan shutil
- Error handling
